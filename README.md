# SETTING UP

## Getting the dotfiles

Clone this repo
````
git clone https://github.com/LoukasN/.dotfiles
````
Install stow
````
sudo pacman -S stow
````
Set up the dotfiles using the install script
````
cd .dotfiles
./install.sh
````
