# CustomConfigs

# **WIP

<p>Configuration files for kitty, rofi, i3, i3status, neovim.
A script that automatically installs dependancies (Only with <code>Pacman</code>).</p>

## How to use

# **WIP

```bash
cd CustomConfigs/CustomScripts/
bash install.sh
```
